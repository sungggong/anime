# AniPick - 애니 추천 랜딩 사이트

HTML, Tailwind CSS CDN, 분리된 CSS/JavaScript로 제작한 단일 페이지 애니 추천 웹사이트입니다.

## 구성

- `index.html` - 페이지 구조와 Tailwind CDN 설정
- `style.css` - 다크 모드 비주얼, 카드, 반응형 보조 스타일
- `script.js` - 장르 필터, 기분별 추천, 랜덤 추천, 찜 목록 localStorage 기능

## 실행

브라우저에서 `index.html`을 열거나 로컬 서버를 실행하세요.

```bash
python -m http.server 4173
```

그 다음 `http://localhost:4173`에서 확인할 수 있습니다.
